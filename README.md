# Реализация SharedPtr / WeakPtr

Учебный C++ проект с реализацией собственных аналогов `std::shared_ptr` и `std::weak_ptr`
для типа `std::string`.

Проект написан **без использования стандартных умных указателей** и предназначен для
демонстрации понимания управления временем жизни объектов, подсчёта ссылок и работы
control block.

---

## Основные идеи

- Один `ControlBlock` управляет:
  - количеством `SharedPtr`
  - количеством `WeakPtr`
- Объект удаляется при `shared_use_count == 0`
- Control block удаляется, когда `shared_use_count == 0` и `weak_use_count == 0`
- `WeakPtr` не продлевает время жизни объекта
- `Lock()` безопасно создаёт `SharedPtr`, если объект ещё существует

---

## Реализованные классы

### `ControlBlock`
Хранит:
- счётчик `shared_use_count`
- счётчик `weak_use_count`

Используется совместно всеми `SharedPtr` и `WeakPtr`, указывающими на один объект.

---

### `SharedPtr`

Поддерживает:
- конструктор по умолчанию
- копирование и перемещение
- создание из `WeakPtr`
- `Reset`
- операторы `*` и `->`
- корректное удаление управляемого объекта
